Single File Programming Question
Implement two stacks in an array

Create a data structure twoStacks that represents two stacks. Implementation of twoStacks should use only one array, i.e., both stacks should use the same array for storing elements. Following functions must be supported by twoStacks.

push1(int x) –> pushes x to first stack

push2(int x) –> pushes x to second stack

pop1() –> pops an element from first stack and return the popped element

pop2() –> pops an element from second stack and return the popped element

Implementation of twoStack should be space efficient.

Note: n is always odd.

Example

Input

5

12 45 78 

89 96

Output

78

96

Explanation

5/2 + 1 => 3 (size of stack 1), 5/2=> 2 (size of stack 1). Insert first three values in stack 1  (12,45,78) and remaining (89,96) in stack 2. Then the popped element from two stacks i.e. 78 (topmost element in stack 1) and 96 (topmost element in stack 2) .

Input format :
The first line of the input consists of the value of n.

Next input is the stack elements.

Output format :
The output prints the popped element from two stacks.

Refer sample input and output for formatting specifications.

Sample test cases :
Input 1 :
5
12 45 78 
89 96
Output 1 :
78
96
Input 2 :
7
235 458 965 854
145 256 322
Output 2 :
854
322
