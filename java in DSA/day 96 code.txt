Single File Programming Question
Given an array has distinct elements, convert the given array in range from 0 to n-1. The order that the arrangement , i.e., 0 is placed in place of smallest element, 1 is placed for second smallest element and so on.

Example

Input

6 9 6 5 4 2 10

Output

4 3 2 1 0 5 

Explanation

The smallest number is 2, which is denoted by 0; the next smallest number is 4, which is represented by 1, and so on. Avoid numbering the duplicate values.

Input format :
Given a single line input separated by space, the first input N denotes the number of elements ,get the N number of input.

Output format :
Single line output separated by space print the reduced array.

Code constraints :
Integers only.

Sample test cases :
Input 1 :
6 9 6 5 4 2 10
Output 1 :
4 3 2 1 0 5 
Input 2 :
6 7 6 5 2 4 10
Output 2 :
4 3 2 0 1 5 

ANS:

// You are using Java
import java.util.*;
class Main{
    public static void main(String[] args){
        Scanner sc=new Scanner(System.in);
        int n=sc.nextInt();
        
        int arr[]=new int[n];
        int t[]=new int[n];
        
        
        for(int i=0;i<n;i++){
            arr[i]=sc.nextInt();
            t[i]=arr[i];
        }
        
        Arrays.sort(t);
        
        HashMap<Integer,Integer>m=new HashMap<>();
        
        for(int i=0;i<n;i++){
            m.put(t[i],i);
        }
        
        for(int i=0;i<n;i++){
            System.out.print(m.get(arr[i])+" ");
        }
        
        
    }
}
