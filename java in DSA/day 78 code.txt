Single File Programming Question
There is a remote village in Maharashtra where people believe that every family must have only 5 members. The elaborate rules are as follows:

There must be one elder member who must have exactly two children.

The first child must be a parent to two children and the second child is not a parent. 

A researcher who came to analyze the practices of this village connected these rules with the binary tree concept. So he planned to implement this in code and explore the in-order, pre-order and post-order traversals.

Input format :
Single line input which has 5 integers.

Output format :
Output displays the pre-order, in-order and post-order traversals of the binary tree as shown in sample test cases.

Sample test cases :
Input 1 :
8 6 9 1 3
Output 1 :
Preorder traversal:
8 6 1 3 9 
Inorder traversal:
1 6 3 8 9 
Postorder traversal:
1 3 6 9 8 


ANS:

// You are using Java
import java.util.*;
class Node{
    int val;
    Node l,r;
    
    Node(int val)
    {
        this.val=val;
        this.l=null;
        this.r=null;
    }
}

class Main{
   
    static void pre(Node root)
    {
        if(root !=null){
            
        System.out.print(root.val+" ");
    pre(root.l);
    pre(root.r);
        }
    
    }
    
    static void in(Node root)
    {
        if(root != null)
        {
            
    in(root.l);
        System.out.print(root.val+" ");
    in(root.r);
        }
        
    }
    static void po(Node root)
    {
        if(root != null){
            
    po(root.l);
    po(root.r);
    System.out.print(root.val+" ");
        }
        
    }
    
    public static void main(String[] args)
    {
       
        Scanner sc=new Scanner(System.in);
       int a = sc.nextInt(); 
        int b = sc.nextInt(); 
        int c = sc.nextInt(); 
        int d = sc.nextInt(); 
        int e = sc.nextInt(); 

        Node root = new Node(a);
        root.l = new Node(b);
        root.r = new Node(c);
        root.l.l = new Node(d);
        root.l.r = new Node(e);
        
        System.out.println("Preorder traversal: ");
        pre(root);
        System.out.println();
        System.out.println("Inorder traversal: ");
        in(root);
        System.out.println();
        System.out.println("Postorder traversal: ");
        po(root);
        
    }
}
