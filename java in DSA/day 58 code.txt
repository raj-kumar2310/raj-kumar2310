Single File Programming Question
Arrange in height

Problem statement

Radha, the school teacher was about to arrange the school students in ascending height order in a queue. Write a program implementing the queue concept to arrange the students standing in the queue based on their height.

Input format :
Input consists of the number of students in the queue 'N',

Followed by N number of students' height.

Output format :
Output displays the students' height in ascending order.

Sample test cases :
Input 1 :
7
5 4 2 6 1 3 7
Output 1 :
Sorted queue: 
1 2 3 4 5 6 7 
Input 2 :
5
3 5 7 8 4
Output 2 :
Sorted queue: 
3 4 5 7 8 
Code Size : 1024 kb
Blacklist
[
]


ANS:

// You are using Java
import java.util.*;
class Main{
    public static void main(String[] args){
        Scanner sc=new Scanner(System.in);
        int n=sc.nextInt();
        Queue<Integer>q=new LinkedList<>();
        
        for(int i=0;i<n;i++)
        {
            q.add(sc.nextInt());
        }
        
        List<Integer>l=new ArrayList<>(q);
        
        Collections.sort(l);
        
        q.clear();
        
        for(int i:l){
            q.add(i);
        }
        
        System.out.println("Sorted queue:");
       while(!q.isEmpty())
        {
            System.out.print(q.remove()+" ");
        }
        
        
    }
}
